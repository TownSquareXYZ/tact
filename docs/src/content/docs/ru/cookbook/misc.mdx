## Как исправлять ошибки

Функция `throw(){:tact}` в контракте полезна, когда мы не знаем, как часто выполнять определенное действие.

Она позволяет преднамеренно обрабатывать исключения или ошибки, что приводит к завершению текущей транзакции и отменяет любые изменения состояния, сделанные во время этой транзакции.

```tact
let number: Int = 198;

// ошибка будет вызвана в любом случае
try { throw(36); } catch (exitCode) {}

// ошибка будет вызвана только если число больше 50
try { nativeThrowIf(35, number > 50); } catch (exitCode) {}

// ошибка будет вызвана только если число НЕ РАВНО 198
try { nativeThrowUnless(39, number == 198); } catch (exitCode) {}
```
