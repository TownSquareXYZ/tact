## Как сгенерировать случайное число

```tact
// Объявляем переменную для хранения случайного числа
let number: Int = 0;

// Генерируем новое случайное число, представляющее собой беззнаковое 256-битное целое число
number = randomInt();

// Генерируем случайное число от 1 до 12
number = random(1, 12);
```
