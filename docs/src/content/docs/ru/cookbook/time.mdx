## Как получить текущее время

Используйте метод `now(){:tact}`, для получения текущего стандартного [Unix-времени](https://en.wikipedia.org/wiki/Unix_time).

Если вам нужно сохранить время в состоянии или закодировать его в сообщении, используйте следующую [сериализацию](/book/integers#serialization): `Int as uint32{:tact}`.

```tact
let currentTime: Int = now();

if (currentTime > 1672080143) {
    // необходимое действие
}
```
