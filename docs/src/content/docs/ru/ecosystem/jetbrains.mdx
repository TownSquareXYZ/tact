Поддерживает подсветку синтаксиса Tact в JetBrains IDE версии **2023.** и более поздних. Обратите внимание, что помимо поддержки Tact, он также включает расширенную поддержку языков FunC и Fift блокчейна TON, а также схем TL-B.

Плагин на маркетплейсе JetBrains: [TON Development Plugin](https://plugins.jetbrains.com/plugin/23382-ton)

## Инструкция по установке

1. Откройте JetBrains IDE (IntelliJ IDEA, PyCharm, WebStorm и т. д.)
2. Перейдите в **Plugin Marketplace**, выбрав `File > Settings/Preferences > Plugins`
3. В строке поиска Plugin Marketplace введите "TON Development". Вы увидите раскрывающийся список с расширением, предоставленным `TON Foundation`.
4. Нажмите кнопку **Install** рядом с именем плагина. Дождитесь завершения установки.
5. После установки плагина вам будет предложено перезапустить JetBrains IDE. Нажмите кнопку **Restart**, чтобы применить изменения.
6. После перезапуска плагин TON Development должен быть успешно установлен в JetBrains IDE.

## Устранение неполадок

Если у вас возникли проблемы в процессе установки, обратитесь к [репозиторию плагина на GitHub](https://github.com/ton-blockchain/intellij-ton) для решения и получения дополнительной информации.
