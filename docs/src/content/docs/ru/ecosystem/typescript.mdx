Язык Tact имеет встроенную поддержку библиотек TypeScript [@ton/ton](https://github.com/ton-org/ton) и [@ton/core](https://github.com/ton-org/ton-core). Компилятор автоматически генерирует код для этих библиотек, поэтому вы можете использовать [@tact-lang/emulator](https://github.com/tact-lang/tact-emulator) или [@ton/sandbox](https://github.com/ton-org/sandbox), которые работают поверх них.

## Контракт Tact в TypeScript

Компилятор генерирует файлы с именем `{project}_{contract}.ts` для каждого контракта в вашем [проекте](/ru/book/config#projects), которые содержат готовые к использованию строго типизированные обертки для работы с ним в любой среде на базе TypeScript: для [тестирования](/ru/book/debug), [развертывания](/ru/book/deploy) и т. д.
