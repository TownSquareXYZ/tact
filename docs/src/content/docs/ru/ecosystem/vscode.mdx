Обеспечивает расширенную поддержку языка Tact в Visual Studio Code:

- Подсветка синтаксиса
- Подсветка ошибок
- Фрагменты кода
- Информация при наведении
- Автоматическое заполнение кода для всех переменных, функций, глобальных параметров и уникальных типов Tact
- Автоматическое заполнение кода для всех контрактов / библиотек в текущем файле и всех связанных импортов
- Форматирование

Расширение на маркетплейсе VS Code: [Поддержка языка Tact для блокчейна TON](https://marketplace.visualstudio.com/items?itemName=KonVik.tact-lang-vscode)

## Руководство по установке

1. Откройте Visual Studio Code (сокращенно называемый VS Code).

2. Перейдите к просмотру расширений, щелкнув значок расширений на панели действий сбоку окна. Это выглядит как квадрат внутри квадрата.

3. В поле поиска расширений введите "Tact Language". Вы должны увидеть раскрывающийся список с расширением "Tact Language", предоставленным KonVik. Вероятно, вы увидите похожее расширение, предоставленное TON Community, но оно устарело, и **вместо него мы должны использовать расширение KonVik**.

4. Нажмите на кнопку "Установить" рядом с названием расширения. Дождитесь завершения установки.

5. После установки расширения вам может потребоваться перезагрузить VS Code. При необходимости рядом с расширением будет кнопка перезагрузки. Нажмите на эту кнопку, если она появится.

6. Теперь расширение Tact Language должно быть установлено в вашем VS Code.

## Включение `Форматирования при сохранении` {#format-on-save}

В этом руководстве будут предоставлены инструкции по включению функции Форматирования при сохранении для расширения Tact Language в VS Code с помощью палитры команд и редактирования файла настроек JSON.

1. Введите `Параметры: Открыть пользовательские настройки (JSON)` в палитре команд. Это откроет Ваш файл `settings.json`.

2. Редактирование настроек JSON

   - Вы увидите объект JSON. Мы добавим некоторые свойства к этому объекту, чтобы включить форматирование при сохранении для расширения Tact Language.

   - Добавьте следующие строки внутри объекта JSON:

     ```json
     {
     	"[tact]": {
     		"editor.formatOnSave": true,
     		"editor.defaultFormatter": "KonVik.tact-lang-vscode"
     	}
     }
     ```

   - Это включит форматирование при сохранении (`"editor.formatOnSave": true`) и установит форматировщик по умолчанию для файлов Tact (`"[tact]": {"editor.defaultFormatter": "KonVik.tact-lang-vscode"}`) для расширения Tact Language.

3. Сохранение и закрытие настроек
   - Сохраните файл `settings.json` после добавления этих строк (можно нажать `Ctrl+S`, чтобы сохранить).
   - Закройте вкладку `settings.json` или нажмите `Ctrl+W`.

Теперь расширение Tact Language VS Code должно автоматически форматировать ваши файлы при их сохранении. Если вы не видите, что эти изменения вступили в силу, возможно, вам может потребоваться перезагрузить VS Code.
