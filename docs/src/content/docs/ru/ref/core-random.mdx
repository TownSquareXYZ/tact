Генерация случайных чисел для смарт-контрактов Tact.

## random

```tact
fun random(min: Int, max: Int): Int;
```

Генерирует и возвращает новое псевдослучайное значение без знака [`Int{:tact}`][int] значение `x` в указанном полузакрытом интервале: `min` $≤$ `x` $<$ `max` или `min` $≥$ `x` $>$ `max`, если и `min`, и `max` отрицательны. Обратите внимание, что значение `max` никогда не включается в интервал.

Примеры использования:

```tact
random(42, 43); // 42, всегда
random(0, 42); // 0-41, но никогда 42
```

## randomInt

```tact
fun randomInt(): Int;
```

Генерирует и возвращает новое псевдослучайное беззнаковое $256$-битное [`Int{:tact}`][int] значение `x`.

Алгоритм работает следующим образом: если `r` — это старое значение случайного начального числа, рассматриваемого как массив $32$-байт (путем построения представления с обратным порядком байтов беззнакового $256$-битного [`Int{:tact}`][int]), то вычисляется его `sha512(r){:tact}`. Первые $32$ байта этого хэша сохраняются как новое значение `r'` случайного начального числа, а оставшиеся $32$ байта возвращаются как следующее случайное значение `x`.

Пример использования:

```tact
let allYourRandomBelongsToUs: Int = randomInt(); // ???, это случайно :)
```

  Расширенные функции для работы со случайными числами перечислены на специализированной странице: [Расширенные API](/ref/core-advanced).

:::
