## Предложения по улучшению TON (TEPs)

Основная цель предложений по улучшению TON — предоставить удобный и формальный способ предлагать изменения в блокчейне TON и стандартизировать способы взаимодействия между различными частями экосистемы. Управление предложениями осуществляется с помощью pull request на GitHub, процесс формально описан в [TEP-1](https://github.com/ton-blockchain/TEPs/blob/master/text/0001-tep-lifecycle.md).

Список [объединенных TEP](https://github.com/ton-blockchain/TEPs#merged-teps).
