Предоставляет функции для получения адресов конфигурации и электора.

Чтобы использовать эту библиотеку, импортируйте `@stdlib/config`:

```tact
import "@stdlib/config";
```

## Функции

### getConfigAddress

```tact
fun getConfigAddress(): Address;
```

Получает параметр конфигурации $0$ в виде [`Address{:tact}`][p].

Исходный код:

```tact
fun getConfigAddress(): Address {
    let cell: Cell = getConfigParam(0)!!;
    let sc: Slice = cell.beginParse();
    return newAddress(-1, sc.loadUint(256));
}
```

### getElectorAddress
