Предоставляет функцию для кодирования off-chain ссылки из [`String{:tact}`][p] в [`Cell{:tact}`][cell].

Чтобы использовать эту библиотеку, импортируйте `@stdlib/content`:

```tact
import "@stdlib/content";
```

## Функции

### createOffchainContent

```tact
fun createOffchainContent(link: String): Cell;
```

Кодирует off-chain `link` из [`String{:tact}`][p] в [`Cell{:tact}`][cell].

Исходный код:

```tact
fun createOffchainContent(link: String): Cell {
    let builder: StringBuilder = beginStringFromBuilder(beginCell().storeUint(0x01, 8));
    builder.append(link);
    return builder.toCell();
}
```
