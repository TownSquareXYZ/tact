## 加强统一部落协议（TEP）

TON Enhancement Proposals（TON 改进提案）的主要目标是提供一种方便、正式的方式，以便对 TON 区块链提出修改建议，并规范生态系统不同部分之间的互动方式。 提案管理是通过 GitHub 拉取请求完成的，[TEP-1](https://github.com/ton-blockchain/TEPs/blob/master/text/0001-tep-lifecycle.md) 对该过程进行了正式描述。
